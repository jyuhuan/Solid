# Solid

**Solid** is a library of strongly typed generic data structures and algorithms 
written in Swift.

Highlights:

- `Graph<Vertex, Weight>`
- `SearchProblem<State, Action, Cost>`
- `Searcher<State, Action, Cost>`
- `Monoid<T>`
- `Semiring<T>`
- `BloomFilter<Element>`
- `CountMinSketch<Element>`
